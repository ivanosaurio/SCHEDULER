# FletApp
Renders another Flet app in the current app, similar to HTML IFrame, but for Flet.
## Properties[​](https://flet.dev/docs/controls/fletapp/#properties "Direct link to Properties")
### `reconnect_interval_ms`[​](https://flet.dev/docs/controls/fletapp/#reconnect_interval_ms "Direct link to reconnect_interval_ms")
Delay, in milliseconds, between reconnection attempts.
### `reconnect_timeout_ms`[​](https://flet.dev/docs/controls/fletapp/#reconnect_timeout_ms "Direct link to reconnect_timeout_ms")
Total time to try reconnecting.
### `url`[​](https://flet.dev/docs/controls/fletapp/#url "Direct link to url")
Flet app URL, e.g. `http://localhost:8550` or `flet.sock`.
## Events[​](https://flet.dev/docs/controls/fletapp/#events "Direct link to Events")
### `on_error`[​](https://flet.dev/docs/controls/fletapp/#on_error "Direct link to on_error")
Fires when a connection or any unhandled error occurs.
